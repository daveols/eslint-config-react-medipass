# eslint-config-react-medipass

## Install

```
$ npm install @medipass/eslint-config-react-medipass --save-dev
```


## Usage

`package.json`:

```
"eslintConfig": {
  "extends": [
    ...
    "@medipass/react-medipass",
    ...
  ],
  ...
}
```

`.eslintrc`/`.eslintrc.json`:

```
{
  "extends": [
    ...
    "@medipass/react-medipass",
    ...
  ],
  ...
}
```
